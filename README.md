# 🏠 RentGuard — Умный помощник для арендодателей

Telegram-бот для автоматизации управления арендной недвижимостью. Помогает частным арендодателям и управляющим компаниям вести учёт объектов, контролировать оплату аренды, вести фотоучёт имущества и генерировать отчёты.

---

## 💰 План монетизации: 100 000 ₽ за месяц

### Модель
- **Стоимость:** 500 ₽/мес за объект (квартиру/дом/комнату)
- **Целевая аудитория:** арендодатели с 1-10 объектами, управляющие компании

### Математика
| Показатель | Значение |
|-----------|----------|
| Цена подписки | 500 ₽/мес за объект |
| Среднее объектов на пользователя | 2-3 |
| Нужно активных объектов | 200 |
| Нужно пользователей | ~70-100 |
| Средний чек (LTR первый месяц) | 1 000-1 500 ₽ |

### Откуда взять первых клиентов (бесплатно)
1. **Avito / ЦИАН** — пиши арендодателям, у которых 3+ объявления. Предложи бесплатный тест на 7 дней.
2. **Telegram-каналы** — «Аренда Москвы», «Сдам/Сниму», «Инвестиции в недвижимость». Публикуй кейсы, а не рекламу.
3. **Юр.лица / УК** — ищи управляющие компании в 2ГИС. У них десятки объектов. Одна сделка = 10 000-30 000 ₽/мес.
4. **Отзывы в 2ГИС / Яндекс.Карты** — найди управляющие компании с плохими отзывами, предложи инструмент для порядка.
5. **Ворд-оф-мус** — каждому довольному клиенту дай 1 месяц бесплатно за приведённого друга.

### План первого месяца
| Неделя | Действие | Цель |
|--------|----------|------|
| 1 | Запуск бота, тестирование, доработка | 5 тестовых пользователей |
| 2 | Поиск клиентов на Avito/ЦИАН (50 сообщений/день) | 20 регистраций |
| 3 | Публикации в Telegram-каналах, холодные звонки УК | 50 регистраций |
| 4 | Реферальная программа, отзывы | 100+ активных объектов |

### Дополнительный доход
- **Настройка под ключ** — 5 000 ₽ за интеграцию под бренд УК
- **Экспорт отчётов для бухгалтерии** — 1 000 ₽/мес (можно включить в Pro-тариф)
- **Pro-тариф** — 1 500 ₽/мес (API, белый лейбл, приоритетная поддержка)

## 🚀 Быстрый старт

### 1. Создай бота в Telegram
- Напиши [@BotFather](https://t.me/BotFather)
- Создай нового бота, получи `BOT_TOKEN`
- Включи **Inline Mode** и **Group Privacy** (при необходимости)

### 2. Получи ID администратора
- Напиши [@userinfobot](https://t.me/userinfobot), получь свой `ID`

### 3. Настрой конфигурацию
```bash
cp .env.example .env
```

Отредактируй `.env`:
```env
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
ADMIN_ID=123456789
YOOKASSA_SHOP_ID=ваш_shop_id
YOOKASSA_SECRET_KEY=ваш_secret_key
PRICE_PER_PROPERTY=500
WEBHOOK_HOST=https://your-domain.com
WEBHOOK_PATH=/webhook
```

> **Для начала** можно оставить YOOKASSA пустыми — оплата будет приниматься вручную через поддержку.

### 4. Запуск локально
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python -m bot.main
```

### 5. Запуск через Docker
```bash
docker-compose up --build -d
```

### 6. Лендинг
Файлы лендинга находятся в папке `landing/`. Загрузи их на любой хостинг (GitHub Pages, Netlify, Cloudflare Pages) или открой локально:
```bash
cd landing
python -m http.server 8000
```
